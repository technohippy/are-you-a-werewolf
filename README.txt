設定項目
- 参加人数
- 存在する役職
- 期間

全体の流れ
- メインWave作成
- 人狼ロボ召喚
- 必要ならホストがいろいろ設定変更
- 募集開始
  メインWaveをpublicに、または自分で誘う
- 設定人数集まるとゲーム開始
  （一定人数を超えていればホストが強制的に開始することもできる）
- 各ユーザーに役割を通知するWaveをポスト
- 狼Wave・墓場Waveを作成
- あらすじを兼ねた説明文表示

- 昼：適当にみんな会話
      LIMIT: 開始時刻 + 設定期間1
- 夕：処刑
      終了判定（狼全滅？）
      LIMIT: 開始時刻 + 設定期間2 or 投票終了
- 夜：狼は狼Waveで投票
      予言者は予言者Waveで誰かの正体を見る
      狩人は今晩守る人を決める
      x 霊媒師は今日死んだ人の正体を見る
      LIMIT: 開始時刻 + 設定期間2 or 投票終了
- 昼：狩人が防御に失敗していれば一人死亡
      予言結果表示
      終了判定（狼過半数？）
      適当にみんな会話
      LIMIT: 開始時刻 + 設定期間1


- 投票タイム
  - 処刑
  - 食餌・ガード・占い
  - 投票タイムは投票と結果発表からなる
- 漫談タイム

必要なWave
- メインWave
  コンソールガジェット
  - 次の投票時間など
  - 各ユーザーのステータス
  - 処刑投票
  - ステータス（昼・夜・協議中・処刑・襲撃に応じた画像を表示）
- 墓場Wave
- 人狼Wave
  襲撃投票
- 役割を通知するために各参加者に一つずつ

検討事項
- Wave参加者と人狼参加者は違う
